My Games

In this page you will find the video games I have developed over the years. Currently, I develop my games with Python using the awesome pygame library. All of the source code for these games are publicly available and can be found in my GitHub page.

Yukon Escape

(18 Dec 2020)

My entry for the 2-Buttons Jam 2020. Yukon Escape is a vertical scroller game set in the treacherous and cold Yukon Mountains. Attempt to survive the harsh terrains by quickly doding obstacles, and surviving 'threat events' - such as earthquakes.

Star Fighter

(8 Dec 2020)

Star Fighter is a shoot 'em up game made in the style of retro arcade video games of the 80s. Take control of a valiant star fighter and make battle with hell-themed enemies to try to set a high score.
